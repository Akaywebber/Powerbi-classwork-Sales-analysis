# Jumia Sales Analysis Dashboard

## Project Overview
This project analyzes sales data from an e-commerce dataset inspired by Jumia.  
The goal of the analysis is to identify sales trends, top-performing products, and revenue patterns using Power BI.

## Tools Used
- Power BI
- Data Cleaning and Transformation
- Data Visualization

## Key Insights
- Identified top-performing product categories
- Analyzed monthly sales trends
- Highlighted revenue contributions across different segments
